# Contributing to Veloura Fashion

First off, thank you for considering contributing to Veloura! It's people like you that make the open-source community such an amazing place to learn, inspire, and create.

## Code of Conduct

All contributors are expected to uphold our code of conduct, which focuses on being respectful, inclusive, and professional.

## How Can I Contribute?

### Reporting Bugs
Before creating a bug report, please check that your issue hasn't already been reported in the issues tab.

### Suggesting Enhancements
Enhancement suggestions are tracked as GitHub issues. When creating an enhancement suggestion, please include as much detail as possible.

### Pull Requests
1. Fork the repo and create your branch from `main`.
2. If you've added code that should be tested, add tests.
3. If you've changed APIs, update the documentation.
4. Ensure the test suite passes.
5. Make sure your code lints.
6. Issue that pull request!

## Style Guide

- Follow standard React and ES6+ conventions.
- Maintain consistent indentation (2 spaces).
- Write descriptive variable and function names.
- Keep components focused and reusable.

## Documentation

If your change affects the UI or API, please update the README.md accordingly.

Happy coding!
